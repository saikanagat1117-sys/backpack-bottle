What each piece is for

Vercel backpack-bottle

Hosts the Next.js site. Auto-deploys on every git push to main. Handles the lead-capture serverless function. Custom domain ready.

GitHub saikanagat1117-sys/backpack-bottle

Source of truth. Connected to Vercel via OAuth — every commit triggers a preview or production build. CI-friendly.

GA4 G-FHBQE8QZM9

Primary analytics. Captures every dataLayer event — form_submit, coupon_download, scroll_75, plus auto page_view + Enhanced Measurement (scrolls, outbound clicks, file downloads).

GTM GTM-KSN24TZ2

Tag manager. Loads alongside GA4 for future flexibility — adding Meta Pixel, Google Ads conversion, or any other pixel without touching code.

Microsoft Clarity whxm989emj

Free session recordings + heatmaps. Filter recordings by custom event ("show me sessions that submitted the form"). Detects rage-clicks and dead-clicks automatically.

Vercel Analytics

Top pages, top referrers, devices, countries. Instant — no setup, no scripts beyond @vercel/analytics already wired.

Vercel Speed Insights

Real-user Core Web Vitals (LCP, FID, CLS) from actual visitors. Critical for the "site performance" deck slide.

Consent Mode v2

GDPR-compliant cookie banner. Defaults to denied; flips to granted via gtag('consent','update') on accept. Keeps modeled traffic in GA4 even when users reject.

Brevo Contacts API

Every lead from /api/lead is POSTed to Brevo with attributes (FIRSTNAME, DEPARTURE_CITY, DESTINATION_INTEREST, LOCALE, COUPON_CODE, SIGNUP_DATE). Auto-segments leads.

Brevo Automation — Welcome Series

4-email flow over 60 days. Day 0 welcome + coupon · Day 3 personalized destination · Day 7 social proof · Day 60 expiry reminder. Personalized via merge tags.

Coupon BB50 PDF

Branded one-page PDF generated by scripts/build-coupon.ts using pdf-lib. Served from /public/backpack-bottle-coupon-BB50.pdf + attached to welcome email.

Custom dataLayer events

cta_click · destination_card_click · form_start · form_submit · coupon_download · visit_count · scroll_75 · outbound_click · consent_update
