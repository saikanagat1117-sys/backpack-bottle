What each piece does

Click cards below or nodes above for full detail

Vercel backpack-bottle

Hosts the Next.js site. Auto-deploys on every git push to main. Handles the lead-capture serverless function.

GA4 G-FHBQE8QZM9

Primary analytics. Captures every dataLayer event — form_submit, coupon_download, scroll_75, plus auto page_view.

GTM GTM-KSN24TZ2

Tag manager. For future Meta Pixel + Google Ads conversion tagging without touching code.

Microsoft Clarity whxm989emj

Free session recordings + heatmaps. Filter by custom event. Detects rage-clicks and dead-clicks.

Vercel Analytics

Top pages, top referrers, devices, countries. Instant — no setup beyond @vercel/analytics.

Vercel Speed Insights

Real-user Core Web Vitals (LCP, FID, CLS) for the "site performance" deck slide.

Consent Mode v2

GDPR-compliant cookie banner. Denied default; flips to granted via gtag('consent','update') on accept.

API /api/lead

Vercel serverless function. Validates the form, logs the lead, fires Brevo contact creation, returns BB50 coupon.

Brevo Contacts API

Every lead lands in list 13 with attributes (FIRSTNAME, DEPARTURE_CITY, DESTINATION_INTEREST, LOCALE, COUPON_CODE).

Brevo Automation

4-email flow over 60 days. Welcome (D0) · Personalized (D3) · Social proof (D7) · Expiring (D60).

Coupon BB50 PDF

Branded one-page PDF generated by scripts/build-coupon.ts. In /public/backpack-bottle-coupon-BB50.pdf.

dataLayer events

cta_click · form_submit · coupon_download · visit_count · scroll_75 · outbound_click
